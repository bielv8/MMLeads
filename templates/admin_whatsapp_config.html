Como Configurar
1. Criar App no Meta for Developers

Acesse Meta for Developers e crie um novo app com WhatsApp Business API.

2. Obter Access Token

No painel do seu app, vá para WhatsApp > API Setup e copie o token temporário ou configure um token permanente.

3. Obter Phone Number ID

Na seção WhatsApp > API Setup, copie o Phone Number ID do número que deseja usar.

4. Configurar Webhook

Na seção WhatsApp > Configuration, configure o webhook com:

  • URL: {{ request.url_root }}webhook/whatsapp
  • Verify Token: whatsapp_verify_token_123
5. Subscrever a eventos

Marque as seguintes opções:

  • messages
  • message_status
